# PocketLedger-Expanse-Tracker
A responsive Daily Expense Tracker built with HTML, CSS, JavaScript, and Firebase. Manage daily expenses with authentication, real-time updates, monthly reports, search history, PDF export, and an intuitive dashboard.


### 💰 Daily Expense Tracker

A modern and responsive **Daily Expense Tracker** built using **HTML, CSS, JavaScript, and Firebase**. This application helps users manage their daily expenses efficiently with authentication, real-time database integration, expense history, and PDF report generation.

## 🚀 Features

* 🔐 User Registration & Login (Firebase Authentication)
* ➕ Add New Expense
* ✏️ Edit Existing Expense
* 🗑️ Delete Expense
* 📅 Current Month Expense Dashboard
* 📊 Monthly & Yearly Expense Summary
* 🔍 Search Previous Expenses by Month & Year
* 📄 Download Expense History as PDF
* ⚡ Real-time Firestore Database
* 📱 Responsive User Interface

## 🔥 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Daily-Expense-Tracker.git
```

2. Open the project in VS Code.

3. Configure your Firebase project.

4. Replace your Firebase configuration in `firebase.js`.

5. Run using Live Server.
